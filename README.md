# AI-Powered Sudoku Solver

A modern, interactive Sudoku Solver web application built using HTML, CSS, and Vanilla JavaScript. It leverages the Backtracking Algorithm to solve Sudoku puzzles efficiently and provides a real-time visualization of the AI's thought process.

## Features
- **Interactive 9x9 Grid:** Play and input manually.
- **AI Visualizer:** Watch the backtracking algorithm at work step-by-step with adjustable speed.
- **Instant Solve:** Instantly evaluate and solve even the hardest boards in the background.
- **Dynamic Error Checking:** Automatically validates your grid for row/col/box duplicate entries.
- **Premium UI:** Designed with modern aesthetics—glassmorphism, vibrant soft colors, hover effects, and crisp CSS animations.
- **Sample Puzzles:** Auto-generate Valid Sudoku configurations (Easy, Medium, Hard).
- **Responsive Navigation:** Keyboard arrow capabilities to map the board and perfectly scalable mobile layout.

## How to Run the Project Locally

Since this is built with vanilla web technologies, you have a couple of options:

### Option 1: Direct File Open
The simplest way! Just double-click the `index.html` file provided in this folder, and it will immediately open and run perfectly in your default web browser without any installation required!

### Option 2: Using VS Code Live Server
1. Open this folder `AI GAME` in Visual Studio Code.
2. Hit the "Go Live" button on the bottom right (if you have the "Live Server" extension installed).
3. The app will launch in your browser.

### Option 3: Python local server
If you want to run it dynamically over localhost:
1. Open your Terminal/Command Prompt in this directory.
2. Run `python -m http.server 8000` (Make sure Python is installed).
3. Open your browser and go to `http://localhost:8000`.

Enjoy watching the AI solve the puzzles!
